## Hook

*The sea returns what it claims.*

A fisherman visits the Dragon Palace for three days — and returns to find his village erased by centuries. **Tides of Urashima** is a 2–3 hour stylized JRPG about consequence, memory, and the price of paradise.

## Audio

- **BGM:** Procedural pentatonic score (pad + koto-like melody + soft taiko pulse) generated by `tools/generate_marketing_trailer.py` and cached as `steam/trailer_bgm.ogg`.
- **No voice-over** — on-screen text only (matches game: selective VO only in gameplay, not trailer).
- **Dialect VO (粵語 / 國語)** does not apply to the trailer — subtitles are written `zh-Hant` regardless.

## Regenerate

```bash
# English only (default)
python3 tools/generate_marketing_trailer.py

# One locale
python3 tools/generate_marketing_trailer.py --locale ja
python3 tools/generate_marketing_trailer.py --locale zh-Hant

# All four trailers + shared BGM
python3 tools/generate_marketing_trailer.py --all-locales

# Force new BGM
python3 tools/generate_marketing_trailer.py --all-locales --regen-bgm
```

Requires: `ffmpeg`, `numpy`, illustrations in `docs/pitch/illustrations/`

**After M5 art rebuild:** re-run `--all-locales` if pitch illustrations change.
